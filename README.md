# Interactive Portfolio Website (o ゜▽  ゜)o☆

⫸ This is a fully responsive and interactive personal portfolio website built using **HTML (Intergrated with CSS and JavaScript)**.  
 ⫸ It showcases my skills, projects, certifications, and contact details with a modern UI and smooth user interactions.  ヾ(≧▽≦*)o

---

## Features  (❁´◡`❁)ﾉｼ

- Light and Dark mode toggle 
- Fully responsive (desktop, tablet, mobile)
- Modern UI with glowing effects and smooth animations
- Accordion-style sections for Skills, Certificates, Projects, and Education
- Back-to-top button for better navigation
- Hamburger menu for mobile-friendly navigation

## Deployment  o(*≧▽≦)ツ

The portfolio is deployed using **AWS S3 Static Website Hosting**.

### Deployment Steps (Summary):   (*^▽^*)
1. Create an S3 bucket  
2. Upload `portfolio.html`  
3. Enable static website hosting  
4. Configure bucket policy for public read access  

The website is securely hosted and accessible via an AWS S3 public endpoint.
